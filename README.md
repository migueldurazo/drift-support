# Drift Support

Need help with Drift? Contact: support@yourdomain.com

## FAQ

### Drift can’t hear me
Make sure Microphone and Speech Recognition permissions are enabled in iOS Settings.

### No sounds are playing
Check volume/silent mode and confirm listening is active.

### Language is incorrect
Open Settings in Drift and select English or Español.
